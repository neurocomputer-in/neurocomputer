/// Language

NeuroLang

The Python library. Typed primitives — Neuro, Flow, Plan, Memory, Effect, Budget, Recovery, Agent. Composition operators: | sequential, & parallel, DAG, loop. Categorically grounded; mypy-checked; renderable as string diagrams.

The Authoring Feel

Reads like a sentence.
Runs like a typed program.

Every neuro carries its own contract — types, effects, budget, docstring. Compose them with operators. Plans are first-class values. You can inspect, replay, and diff every run.

From idea to plan in three lines.

Define neuros with declared effect and budget. Compose them with the | operator. Get back a Flow — a structured plan you can render, cost-estimate, and run.

Where the flow is differentiable, it's also a tensor network. Where it's symbolic, it's a string diagram. Same object. No translation.

See the primitives →
# pip install neurolang from neurolang import neuro, Flow, Memory from neurolang.stdlib import web, reason @neuro(effect="tool") def extract_book(url: str) -> dict: """Scrape title, author, summary.""" ... research: Flow = ( web.search | extract_book | reason.summarize | memory.store ) plan = research.plan(query="category theory") plan.cost_estimate() # budget rolled up plan.run(memory=Memory.discrete()) plan.serialize() # inspect, replay
The Hierarchy

Agency. Project. Agent. Neuro.

Four layers. Each groups the next. The runtime is built so you can scale a thought from a single capability to an entire agency without changing the model.

L1

Agency

Top-level workspace — a team. Color, emoji, set of agents, default project. Neuro HQ, Upwork, Web Agency.

L2

Project

A unit of work inside an agency. Default project ships with every agency. Boundaries for memory and conversations.

L3

Agent

Role-specialised orchestrator with router + planner + replier + profile. neuro, nl_dev, opencode, openclaw.

L4

Neuro

The atomic unit. conf.json + code.py + optional prompt.txt. Hot-loadable, validated, composable.

What You Get

A complete substrate.

Everything wired together — voice, vision, multi-agent rooms, scheduler, mobile remote. One runtime, many surfaces.

Voice

Voice agent over LiveKit

STT/TTS round-trip with Sarvam streaming + ElevenLabs / OpenAI synthesis. Runs as a system service. Agents speak, listen, and converse with sub-second latency.

3D IDE

Neurogramming graph IDE

Author neuros visually with @react-three/fiber. Source-aware editing of conf.json, code.py, prompt.txt. Hot-load on save.

Multi-Agent

Meeting Rooms

Multiple agents share a transcript with a mediator picking the next speaker round-robin. Live in room_* neuros + RoomPanel.tsx.

Scheduler

Time-aware triggers

Cron-style triggers persisted to schedules.db via APScheduler. Set timers, recurring digests, deferred tasks — all from the chat surface.

agent.talk

Agent-to-agent

Direct typed messages between agents with depth-guard (MAX=4). The substrate beneath rooms — and the seed of categorical mailbox protocols.

Profiles

Behaviour packs

Swap model, system prompt, defaults at the agent level. Ships with general, code_dev, neuro_dev, neurolang_dev.

Mobile

Android remote

Jetpack Compose client. Voice typing. Remote desktop with WebRTC streaming. Pinch zoom up to 10×. Multi-monitor switching.

NL Dev

NeuroLang authoring agent

Describe a flow in natural language. The nl_dev agent compiles it into a runnable neuro and saves it under ~/.neurolang/neuros/.

Why It's Different

Architecture, not features.

Most agent frameworks treat the LLM as opaque and the program as untyped pipelines. NeuroLang treats both as typed compositional structure. Effects in types. Plans as values. Memory with scope. Recovery as a language primitive.

Quickstart

Run it locally in five steps.

Single Python core, vendored NeuroLang library, Next.js web, optional LiveKit voice, optional Android.

git clone git@github.com:neurocomputer-in/neurocomputer.git
cd neurocomputer
python3 -m venv .venv && source .venv/bin/activate
pip install -r neurocomputer/requirements.txt
pip install -e ./neurolang   # vendored framework

python neurocomputer/server.py             # http://127.0.0.1:7000
cd neuro_web && npm install && npm run dev # http://localhost:3000
