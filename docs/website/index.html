The Vision: Agency AI

The pathway to AGI.

Computing is undergoing a fundamental shift. We are advancing through clearly defined epochs of reasoning and automation, moving beyond monolithic models into hierarchical intelligence.

1. Software → Hardcoded logic and explicit, rigid rules.
2. Statistical ML → Probabilistic pattern recognition and classification.
3. Generative Models → Transformers predicting structured outputs and tokens (LLMs).
4. Reasoning Models → Systems that maintain state, plan, and execute Chain-of-Thought (ReAct).
5. Agentic Systems → Autonomous loops that observe, reason, and act across domains.
6. Agency AI (The Neuro Vision) → Hierarchical, modular agencies consisting of shared agents. Abstract, modifiable, and capable of infinite scaling.

Practical Applications (The OS Level)

Well-tested OS ecosystem apps.

At the practical layer, Neuro operates like an Operating System. Our agents function as highly specialized "apps" with their own runtimes, seamlessly available on demand.

Core Runtime

Neuro

The general-purpose core network and runtime. It reasons about intent, plans tasks natively via graphs, and routes execution to specific applications within the agency.

App

OpenClaw

The browser operator app. Connects to a local gateway to physically browse the web, interacting directly with DOM elements and managing automated web workflows.

App

OpenCode

The developer's companion app. Reads entire repositories natively, generates complex code diffs, patches systems, and runs deep technical analysis as a background task.

App

NeuroUpwork

The workflow application. It hunts for job listings, assigns numerical and entropy-based fit scores, and generates precise proposals and proof-of-concept plans.

Under the Hood

The Brain architecture.

A Reason + Act loop that classifies, plans, and executes — keeping simple things fast and complex things structured.

Smart Router

Every message hits the router first. In a single LLM call, it classifies the request: direct reply, or skill execution? If a skill is needed, it identifies which one.

Simple questions get answered instantly. Complex tasks get planned and executed step by step. No unnecessary overhead.

User Input │ ▼ Router ─── "just chat?" ─── Direct Reply │ ▼ needs a skill Planner ─── builds DAG │ ▼ Executor ─── Skill A → B → C │ ▼ Reply ─── Final Answer

Skills Are Modular

Each skill lives in its own folder: a conf.json for metadata and a code.py for logic. Drop a new folder in, and the Brain discovers it at runtime. No restart needed.

You can even create and edit skills at runtime using meta-skills — the framework can modify itself.

skills/ ├── lock_screen/ │ ├── conf.json │ └── code.py ├── read_file/ │ ├── conf.json │ └── code.py ├── take_screenshot/ │ ├── conf.json │ └── code.py └── [your_skill]/ ├── conf.json └── code.py

Secure Sandbox

Neurocomputer is designed to run inside a private, isolated machine — a dedicated workstation, a VM, or a container that is separate from your personal host OS.

Autonomous agents need access to your screen, files, and inputs. By isolating this to a dedicated environment, you control exactly what they can see and touch. This is the recommended deployment model.
